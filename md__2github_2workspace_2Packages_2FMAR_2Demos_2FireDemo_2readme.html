FMAR Demo - Fire

The FMAR Fire Demo is an example of how many of the different modules can be combined into a cohesive experience. The fire demo showcases many modules that would be useful during a fire rescue operation, as well as additional modules and scripts to help set the scene for such an event. These include:

  • Magio (Unity Asset Store Purchase)
    • Magio is used to assist in creating the smoke and fire effects critical to the demo. While these effects have been heavily modified and supplmented to fit our needs, Magio still remains a dependency of the demo.
    • This is the only piece of FMAR to require an asset store purchase. Due to the license of this asset we cannot re-distribute it.
  • Fire Controls
  • Air + (Air Gauge & Air HUD)
  • Draggable Anchors
  • Gestures
  • Indicators
  • Pathfinder
  • Map
  • Points of Interest
  • Radial Menu
  • Tabbed Display
  • Notifications
  • Peripheral Indicators / Fader

Note that this demo is designed for use with the Meta Quest 3 headset, for various reasons. The smoke and fire simulation is ineffective on optical see-through devices, as it relies on reducing the users vision which can only be done to a very limited extent on this type of hardware. In addition, the additional performance and depth information provided by the Quest 3 headsets are important for rendering such an advanced effect on a standalone headset. Access to the quest's experimental depth API is also required for this demo to function properly.
